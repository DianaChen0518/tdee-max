# TDEE Tracker 代谢计算引擎技术协议 (Metabolic Protocol)

本文件详细记录了 TDEE Tracker 内部所采用的所有生理代谢计算公式、科学依据及逻辑推导过程。

---

## 1. 基础代谢率 (BMR)
采用 **Mifflin-St Jeor** 公式，这是目前公认对现代人群最准确的估算模型。

### 计算公式
- **男性**: $BMR = (10 \times W) + (6.25 \times H) - (5 \times A) + 5$
- **女性**: $BMR = (10 \times W) + (6.25 \times H) - (5 \times A) - 161$

> **变量定义**:
> - $W$: 当前体重 (kg)
> - $H$: 身高 (cm)
> - $A$: 精确年龄 (Age)

### 年龄计算逻辑 (Continuous Age)
系统采用“日期相关年龄”算法，而非简单的足岁整数：
- **基准**: 基于 **365.25 天/年**（考虑闰年）。
- **逻辑**: 计算历史记录时，会回溯至记录当天的精确年龄（如 25.43 岁），确保 BMR 追溯的科学性。

---

## 2. 非运动活动产热 (NEAT)
基于步数和自身体重的功耗模型。

### 计算公式
$NEAT = Steps \times (0.04 + 0.0005 \times W)$

- **0.04**: 基础步数热量系数。
- **0.0005**: 体重代偿系数（考虑克服重力所需的额外功）。

---

## 3. 运动活动产热 (EAT) - 净能耗模型
**核心原则**: 采用“净能耗 (Net Energy Expenditure)”逻辑，扣除运动时段内重复计算的 BMR。

### A. 有氧运动 (Keytel 模型)
采用 **Keytel et al. (2005)** 能量消耗预测模型。
- **公式 (毛能耗)**:
  - **男**: $(-55.0969 + 0.6309 \times HR + 0.1988 \times W + 0.2017 \times A) / 4.184$
  - **女**: $(-20.4022 + 0.4472 \times HR - 0.1263 \times W + 0.0740 \times A) / 4.184$
- **公式 (净能耗)**:
  $Net\_EAT = Gross\_EAT - (BMR / 1440) \times Duration$

### B. 无氧/阻力训练 (MET 模型)
- **公式**:
  $Net\_EAT = (MET - 1) \times W \times (Duration / 60)$
- **强度对照**:
  - 高强度: 7.0 METs (Net 6.0)
  - 中强度: 5.0 METs (Net 4.0)
  - 低强度: 3.5 METs (Net 2.5)

---

## 4. 后燃效应 (EPOC)
针对高强度有氧运动后的额外氧债补给计算。

### 关键前置指标
- **最大心率 (Tanaka 公式)**: $Max\_HR = 208 - 0.7 \times A$
- **心率储备百分比 (%HRR)**: $Intensity = (Avg\_HR - RHR) / (Max\_HR - RHR)$

### 核心计算函数
$EPOC = Net\_EAT \times Intensity\_Ratio \times Duration\_Factor$

#### 1. 强度系数 (3段线性模型)
| %HRR 范围 | 计算公式 (Ratio) |
| :--- | :--- |
| < 0.60 | $0.03 + 0.05 \times Intensity$ |
| 0.60 ~ 0.80 | $0.06 + 0.25 \times (Intensity - 0.60)$ |
| ≥ 0.80 | $0.11 + 0.20 \times (Intensity - 0.80)$ |

#### 2. 时长修正因子 (Duration Factor)
| 时长 (T) | 修正系数 |
| :--- | :--- |
| T < 15min | $0.7 + 0.3 \times (T / 15)$ |
| 15min ≤ T < 40min | $1.0 + 0.1 \times ((T - 15) / 25)$ |
| T ≥ 40min | $1.1$ (封顶项) |

---

## 5. 食物热效应 (TEF)
$TEF = BMR \times 10\%$
(注: 采用 BMR 基数的 10% 作为混合膳食的平均消化能耗支出)。

---

## 6. 总能量消耗 (TDEE) 总公式
系统每日结算的总能量支出的顶层逻辑：

$$TDEE = BMR + TEF + NEAT + Net\_EAT + EPOC$$

---

## ⚖️ 精度与免责声明
- **计算误差**: 基于群体统计学模型，个体实测值可能存在 **±25%** 的偏差。
- **临床局限**: 本公式不适用于孕妇、极高体脂率人群或具有特定代谢疾病的个体。
