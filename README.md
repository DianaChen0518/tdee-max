## 💡 为什么是 TDEE MAX？

市面上的大多数 TDEE (Total Daily Energy Expenditure) 计算器通常采用简单的静态系数去预测个人的日常消耗。这种粗略的平均数不仅容易导致减脂或增肌过程中的停滞，还可能造成饮食摄入的长期失衡。

**TDEE MAX** 是一款使用现代 Web 技术栈构建的专业级追踪看板。我们不仅提供了一个**遵循严格网格规范与人体工程学**的数字座舱级界面，更在底层搭载了一套**由运动生理学驱动的动态代谢计算引擎**。我们主张“净消耗”理念，不重复计算底噪，不盲目乐观，为你提供一个最真实的身体能量账本。

## 🧬 核心代谢引擎 (Metabolic Engine)

本项目的灵魂在于抛弃了传统的“拍脑袋估算”，我们通过解析并运用行业权威模型，基于个体的真实活动数据进行了高度自定义的连续推算：

- **精确基础代谢 (BMR)**：采用对现代及都市人群最为精准的 **Mifflin-St Jeor** 公式，结合日历级随时间生长的“精确连续年龄”，彻底杜绝因“长了一岁”导致的卡路里阶跃断层。
- **运动净效产热 (Net EAT)**：严苛的“净能耗”计算标准，自动从运动时长中扣除这期间本就该属于 BMR 的份额。有氧运动根据 **Keytel (2005)** 结合心率推演，抗阻力训练通过梯次 **MET** 模型计量。
- **真实后燃效应 (EPOC)**：引入 **Tanaka** 最大心率公式并映射心率储备百分比 (%HRR)。我们自己搭建了一个三段线性模型，科学地补偿那些让心跳飙升的高强度训练在运动后产生的额外氧债消耗。
- **日常活动与进食热效应 (NEAT & TEF)**：用算法修正了基础步数对应的热量折损，当监测到有效饮食摄入时，优雅地释放 10% 的 TEF 转化算力。

> **🔒 引擎防线 (Negative Defense)**: 业务逻辑层引入了严格的值域钳制与健康度检查。针对部分外接手环或心率带可能产生的“漂移异常值”进行平滑截断，确保算法产生的能耗绝不倒扣、亦不造假。

## 🛠 项目架构与技术栈

在这个项目中，我们遵循了极致的模块化设计与领域驱动理念 (DDD)。所有的计算逻辑都已被抽离到独立的 Service 或 Hook 层，让 UI 保持绝对纯粹的功能性和高吞吐量响应：

### 高性能技术栈
- **核心框架**: Vue 3 (Composition API) + Vite (极速冷热启动)
- **类型系统**: TypeScript (严谨、强类型的声明护航)
- **状态中枢**: Pinia (配合完善的本地持久化与状态拆分)
- **唯美视觉**: Tailwind CSS (支持原生丝滑暗黑模式、高阶流体排版)
- **数据呈现**: Chart.js 配合 vue-chartjs (轻量级图表驱动)

### 源码结构透视
```text
src/
├── components/       # 模块化 UI 组件 (高度解耦)
│   ├── features/     # 核心功能域 (tdee / workout / diet 等)
│   └── layout/       # 脚手架与布局 (Header / Notification等)
├── composables/      # 响应式组合函数 (如全局消息队列、主题切换)
├── constants/        # 协议、计算常数字典与公用魔法数字
├── i18n/             # Vue-i18n 全局多语言词典
├── services/         # 独立业务或复杂计算引擎 (如 ReportingService)
├── store/            # 应用级数据中枢 (承接云端与本地缓存)
├── utils/            # 算法外延、文件 I/O 汇出及纯工具集
└── types.ts          # TypeScript 全景数据结构及 Interface
```

## ✨ 核心功能亮点

- **📐 专业座舱面板**：采用了经过多次打磨的三列式信息流设计，遵循 8px 网格，信息密度极高但不显压抑。
- **📊 动态能量闭环**：不仅记录枯燥的数字，还会通过 `EnergyRing` 等定制化组件将卡路里的消耗与盈亏直观可视化。
- **☁️ 云端同步与本地接管**：既可以在本地毫秒级响应确保无网可用，也提供了可靠的异步云端备份方案（带完善的状态推送）。
- **🌍 原生多语言架构**：提供了完整、零侵入式的 i18n 支持。
- **📈 深度数据审计**：不仅仅是报表，还支持将最硬核的历史数据完整格式化导出为 `.xlsx` 格式，把数据的掌控权百分之百交还给你自己。
